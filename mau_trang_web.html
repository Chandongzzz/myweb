Bài tập: Tạo trang web đơn giản

Yêu cầu: trang này có ít nhất 3 ảnh, danh sách 3 mục và một liên kết. Dùng file này làm đề bài để code lại hoặc mở rộng.

Thư viện ảnh (ít nhất 3 ảnh)

Danh sách việc cần làm (3 mục)

  • Viết HTML cơ bản: header, section, footer.
  • Thêm CSS để làm đẹp (gợi ý: border-radius, box-shadow).
  • Thêm JavaScript nhỏ (ví dụ: sự kiện click thay đổi ảnh).

Nếu cần tham khảo hoặc nộp bài, mở liên kết bên dưới:

Mở liên kết mẫu

Gợi ý: thử thay src của ảnh, thay nội dung danh sách, hoặc thêm form nộp bài.
